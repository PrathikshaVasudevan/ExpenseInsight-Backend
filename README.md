# ExpenseInsight Backend Engine

A Python backend engine built for hackathon-style expense analysis.

## Features
- Data cleaning & validation
- Total spending calculation
- Category-wise analysis with percentage
- Weekly spending comparison
- Overspending alerts

## Tech Stack
- Python
- Pandas

## Project Structure
backend/
analysis.py
alerts.py
main.py
data/
expenses.csv

```bash
python backend/main.py
